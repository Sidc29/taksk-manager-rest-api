Get started

The Task Manager REST API is a web-based API that allows clients to interact with a Task Manager application over the internet. It provides a set of endpoints that can be used to manage tasks, such as creating, reading, updating, and deleting tasks. The API uses standard HTTP methods, such as GET, POST, PATCH, and DELETE, to perform these operations.

Check out the demo - https://task-manager-api-hnef.onrender.com/api/v1/tasks

Use a JSON viewer extension/plugin to best view the JSON data

how to use

To get all the tasks you need to make a GET request to the following url :
https://task-manager-api-hnef.onrender.com/api/v1/tasks

To create a task you need to make a POST request to the following url :
https://task-manager-api-hnef.onrender.com/api/v1/tasks

To get a single task you need to make a GET request to the following url :
https://task-manager-api-hnef.onrender.com/api/v1/[TASK_ID]

To update a task you need to make a PATCH request to the following url :
https://task-manager-api-hnef.onrender.com/api/v1/[TASK_ID]

To delete a task you need to make a DELETE request to the following url :
https://task-manager-api-hnef.onrender.com/api/v1/[TASK_ID]
